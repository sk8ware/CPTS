## 📖 Descripción

Este repositorio documenta mi preparación para la certificación **Certified Penetration Testing Specialist (CPTS)** de **Hack The Box Academy**.

El objetivo es recopilar apuntes, metodología, cheatsheets, herramientas, laboratorios y recursos utilizados durante el estudio, manteniendo un enfoque práctico orientado a pentesting profesional.

> **Nota:** Este repositorio no contiene flags, respuestas del examen ni información que viole los términos de Hack The Box.

---

# 🎯 Objetivos

- Completar todos los módulos de la ruta CPTS.
- Documentar metodologías de pentesting.
- Crear cheatsheets reutilizables.
- Automatizar tareas repetitivas con Python y Bash.
- Mejorar el proceso de elaboración de reportes técnicos.
